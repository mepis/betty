# voice-agents

Voice agents are AI systems that interact with users through spoken conversation. They combine STT, LLM inference, TTS, and turn-taking detection to enable real-time voice conversations. Voice agents are deployed through API platforms (OpenAI Realtime, Google Gemini Live, ElevenLabs), voice agent platforms (Retell AI, Vapi), and custom-built cascaded pipelines.
