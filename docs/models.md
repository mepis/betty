# Models

The Models tab lets you discover, browse, and download GGUF models from HuggingFace.

See also: [[USER-MANUAL]]

## Search Models

1. Enter a search query (e.g., `llama 3`, `mistral`, `qwen`)
2. Optionally check **GGUF only** to filter for GGUF-format models
3. Click **Search**

Results show:
- Model name and description
- Download count and likes
- Last modified date
- Tags (GGUF, PyTorch, etc.)

## Model Details

Click any model to see:
- Full description
- Statistics (downloads, likes, last updated)
- All tags
- Link to the HuggingFace page

## Downloading Models

1. Click a model to open its details
2. Select a `.gguf` file from the file list
3. Click **Download**
4. Watch the progress bar

Downloads are saved to the `hf_downloads/` directory.

## Downloads Tab

View all downloaded models:
- List of model repositories with their downloaded files
- File sizes shown for each file
- Click the **trash icon** to delete a downloaded model
